# Powder River Media — Logo File Reference
*All files located in: Data Dump/PRM Logo/Horizon + Wave Logo/*

## Logo Formats Available
- **SVG** — Scalable, use for web, print, anything digital. Always prefer SVG.
- **PNG @ 600px / 900px / 1800px** — Use when SVG isn't supported (social media uploads, some email clients)
- **favicon.ico** — Browser tab icon, already sized correctly

## Three Lockup Styles
| Style | Use When |
|-------|----------|
| **Horizontal** | Nav bars, email headers, letterheads, banners |
| **Stacked** | Square formats, social posts, print collateral |
| **Vertical** | Tall/narrow formats, posters, signage |

## Icon Only
- Use as social media profile photo (1080px versions available)
- Use as favicon alternative
- Use when wordmark isn't needed (watermarks, small placements)

## Color Variants — When to Use Each
| Variant | Background | Best For |
|---------|------------|----------|
| `aqua-on-black` | Black | Dark backgrounds, premium feel |
| `aqua-on-dark-teal` | Dark Teal `#1D2B2E` | Primary brand dark bg |
| `aqua-on-white` | White | Light backgrounds, print |
| `aqua-on-transparent` | Transparent | Overlays on photos |
| `coral-on-black` | Black | Alternate warm accent |
| `coral-on-dark-teal` | Dark Teal | Alternate warm on dark |
| `coral-on-white` | White | Alternate warm on light |
| `coral-on-transparent` | Transparent | Alternate overlay |
| `bw-dark-on-white` | White | Single-color print, B&W |
| `bw-dark-on-transparent` | Transparent | B&W overlay |
| `bw-white-on-black` | Black | Reversed B&W |
| `bw-white-on-dark-teal` | Dark Teal | Reversed B&W on brand dark |
| `bw-white-on-transparent` | Transparent | White overlay |

## Watermarks
Available in horizontal and vertical, 900px PNG + SVG. Use for:
- Photo watermarking
- Video lower-thirds
- Document footers

## Primary / Default Logo
**For the website:** `aqua-on-dark-teal` (matches the site's dark background)
**For light backgrounds:** `aqua-on-white`
**For social profile photo:** `PRREM-social-profile-aqua-on-black-1080px.png`

## Coral Color Note
Coral (`#E8826A`) is an alternate accent color with full logo variants available.
It is not part of the primary brand palette currently used on the website.
Consider for: seasonal campaigns, secondary marketing materials, A/B testing.
